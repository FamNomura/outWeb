ポンプ車操法における1番員は、主に第1線の筒先および第3ホースの搬送・延長、そして火点への注水（放水）と余裕ホースの確保を担う、攻撃の要となる役割を持っています。

実施手順と注意点について、時系列に沿って詳細に整理します。

1. 待機・点呼・乗車・下車

  • 待機: 待機位置（車両助手席側後部ドア横の乗車しやすい位置）に、火点方向を向いて整列休めの姿勢で待機します。
  • 点呼・乗車: 指揮者の「乗車」の号令で、そのまま右向け右をして速やかに乗車します（車両への乗り込みは三点支持を励行）。乗車後は車両の走行に対応できる姿勢をとります。
  • 下車: 指揮者の「操作始め」の号令の後、4番員の「よし」の合図を確認してからドアを開放して下車します。
  • 【注意点】: 4番員の呼唱前に行動を開始すると大きな減点（3点）となります。また、下車時のドア開放は窓から目視で後方確認後、二段操作で行い、勢い任せに閉めず最後まで片手を離さないようにします。

2. 第1線延長（メインの役割）

  • 筒先と第3ホースの搬送:
  • 下車後、ポンプ車左側に積載してある筒先を取り外し背負います。
  • 積載ホース（第3ホース）のめす金具部を右手で、反対側を左手で持ち、左肩に乗せます。
  • めす金具部を左手に持ち替え、右手を下ろして左に向きを変え、ポンプ車左側を通って火点に向かって前進します。
  • ホースの展張と筒先の結合:
  • 第3ホース展張地点に至ったら、右手でめす金具部を持ち替え、左手で反対側を保持し、めす金具が手前になるように肩から下ろして地面に立て、前方へ転がして展張します。
  • 右手でおす金具を持ったまま左足を軸に右回りに反転し、おす金具を左足近くに置きます。
  • 身体を起こして筒先を第3ホース（おす金具）と結合し、確認します（結合確認はハカマ部分を引いて行います）。
  • 左手でプレイパイプ上部、右手で取手を握り、火点側へ1歩踏み込んで基本注水姿勢をとります。
  • 放水開始の合図と前進:
  • 2番員が第2ホースを第3ホースに結合し姿勢を正したのを確認したら、2番員に対し「放水始め」と呼唱します。
  • 2番員の復唱後、火点に向かっておおむね15メートル前進します。
  • 余裕ホースの確保と注水:
  • 放水停止線の手前で、左上腕と腹部で筒先を抱え込み、折りひざ（またはそれに準じた姿勢）で右手でホースを手繰り寄せます。
  • 右手でホースを持ったまま立つと同時に右足を1歩大きく踏み出し、半円を描くように広げておおむね5メートルの余裕ホースをとります（後方に約1mの注水補助場所を作ります）。
  • 右手を滑らせて筒先を右脇下と右腕で完全に抱え、左手でノズルを徐々に開きます。
  • ノズルが開いたら、左手をプレイパイプ上部に、右手を取手に滑らせて握り、標的に向かって注水します。

3. 第2線延長

  • 指揮者の「第2線延長始め」の号令に対し、これを復唱したのち、そのまま注水操作を継続します。

4. 放水中止

  • 放水停止: 指揮者の「放水止め」の号令で、基本注水姿勢からノズルを徐々に閉じて放水を中止し、「第1線放水止め」と呼唱します。
  • 排水操作:
  • 3番員が注水部署に戻り「伝達終り」の合図を出した後、筒先を右脇下にしっかり抱え、ノズルを徐々に開いた後、**ノズルを下方に向けて排水（下向きで全開）**します。
  • 合図によりノズルを完全に閉めます。
  • 左手をプレイパイプ上部、右手でノズルを握って「よし」と呼唱し、左足を右足に引きつけて筒先を右足ぎわに立てて姿勢を正します。

5. 収納・点検・解散

  • 収納:
  • 指揮者の「おさめ」の号令に「よし」と呼唱し、筒先をホースから離脱して背負います。
  • 指揮者とともに発進し、延長されたホースをまたぎながらポンプ車左側を通り、筒先を元の位置に収納して待機位置に集まります。
  • 点検・解散:
  • 身体や服装の点検を行い、指揮者の「点検報告」の号令で「1番員異常なし」と報告します。
  • 終了報告中は基本の姿勢で待ち、「わかれ」の号令で挙手注目の敬礼を行い解散します。

1番員における特に重要な注意事項（審査・減点ポイント）

  1. 筒先位置（放水停止線）の厳守: 放水停止線を越えて筒先を構えたり足を踏み越えたりすると、3点の大きな減点となります。
  2. 余裕ホースの確保: おおむね5メートルの余裕ホースをとり、2番員が注水補助を行うスペース（後方おおむね1m）を適切に確保できていないと減点対象となります。
  3. 注水姿勢の安定: 放水中の「ふらつき」（右手が腰部から離れる、足の踏み換えが生じるなど）や、筒先からの放水方向が上下左右に1メートル以上ぶれた場合は減点（注水姿勢不安定）となります。
  4. ホース展張と排水操作の要領: ホース展張時は身体と左足先を火点側に向ける必要があります。また、排水操作終了後は、右手でノズルを横から握り、右足ぎわに置いて立ち上がるなど、正しい要領で行わないと減点対象です。
