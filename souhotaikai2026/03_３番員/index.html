ポンプ車操法における3番員は、第1線延長時に吸管の伸長・投入およびとび口の搬送・構えを行い、第2線延長時には第2線のホース（第1・第2ホース）の延長と放水開始の伝達を行う、前半と後半で役割が大きく切り替わるポジションです。

実施手順と注意点について、時系列に沿って詳細に整理します。

1. 待機・乗車・下車

  • 待機: 待機位置（車両運転席側後部ドア横の乗車しやすい位置）に、火点方向を向いて整列休めの姿勢で待機します。
  • 点呼・乗車: 指揮者の「乗車」の号令で、そのまま左向け左をして速やかに乗車し、車両の走行に対応できる姿勢をとります。
  • 下車: 指揮者の「操作始め」の号令の後、4番員の「よし」の合図を確認してから、ドアを開放して下車します。

2. 第1線延長（吸管伸長と、とび口の搬送・構え）

  • 吸管伸長・投入:
  • 下車後、ポンプ車後方へ向かい吸管積載部の左側に「右向け止まれ」の要領で止まり（開脚）、近くの吸管止め金をはずします。
  • 4番員から送り出されたストレーナー付近の吸管を両手で受け取り、吸管の左側に出て、右手は下から、左手は吸管を左腋下にかかえるように持ち替えます。
  • 4番員の「よし」の合図で歩調を合わせ、左足から2歩半で吸管投入位置まで進み、吸管をその場に置きます。
  • 右足を立てた折りひざの姿勢で吸管控綱を取り外し右脇に置きます（控綱を固定しておく輪ゴムはそのまま籐かご等に付けたままにします）。
  • 右手で控綱の根本と端末、左手で吸管を持って立ち上がり、「よし」と合図して右足を半歩前に出し、4番員の協力で吸管を水利に投入します。
  • 控綱の結着と枕木配置:
  • 吸管控綱の端末を右手で持ち、ポンプ車方向へかけ足で発進します。ポンプ車後部のきょう固な位置に、もやい結び及び半結び（端末をおおむね10cm残す）で結着します。
  • 枕木を取り出し、吸管投入位置へかけ足で戻り、枕木を吸管の下に敷いてバンドを取り付けます。
  • とび口の搬送と部署:
  • とび口積載部に向かい「右向け止まれ」で止まり、とび口をおろして左腋下に抱え、延長ホースの左側に沿って破壊地点までかけ足で向かいます。
  • 左手で柄の中央部、右手で柄の後部（後端からおおむね10cmを残した位置）を持ってとび口を構えます。

3. 第2線延長（ホース延長と放水開始の伝達）

  • ホースの搬送と展張・結合:
  • 1番員の「第2線延長始め」の復唱に対して「よし」と呼唱し、とび口をその場に置いてポンプ車後部のホース積載位置へ向かいます。
  • 第2ホースを搬送に便利な位置に置き、第1ホースをめす金具が手前になるように展張し、おおむね2メートルの余裕ホースをとって第2放口に結合します。
  • 第2ホースを左肩にかつぎ、第1ホースのおす金具を右手に持ち、展張ホースの左側に沿って延長します。
  • 第2結合場所で第1ホースを置き、第2ホースを展張して結合します。同様に第2ホースのおす金具を持って延長し、第3結合場所で第3ホースと結合します。
  • 火点の方向に向きを変え、右足を左足に引きつけて基本の姿勢をとります。
  • 放水開始の伝達と注水部署:
  • 2番員の「第2線放水始め」の合図を復唱し、回れ右をしてポンプ車方向へ発進します。
  • 第2結合部付近をまたいで伝令停止線内の位置で4番員に相対して停止し、右手を垂直にあげて「第2線放水始め」と伝達します。
  • 4番員が復唱の手を下ろした後、回れ右をして最短距離で第2線の注水部署（2番員の反対側1歩後方）に至り「伝達終り」と呼唱します。その後、再び破壊地点に至り、とび口を構えます。

4. 放水中止

  • 伝達と部署:
  • 1番員と2番員の「放水止め」の合図を確認後、「よし」と呼唱してとび口を置き、第1線の延長ホースの左側に沿って伝令停止線内まで移動します。
  • 4番員に相対し、右手を横水平にあげて「放水止め」と伝達します。
  • 回れ右をして、第1線の延長ホースの左側に沿って注水部署（第1線と第2線の注水部署の中間地点）に停止し、「伝達終り」と呼唱します。
  • 左足を1歩踏み出し、とび口をとり右足ぎわに立てて姿勢を正します。

5. 収納・点検・解散

  • 収納:
  • 指揮者の「おさめ」の号令に「よし」と呼唱し、とび口を浮かし、回れ右の要領で向きを変えると同時に左腋下に抱えます。
  • 第2線第1ホースをまたいでとび口収納位置に至り、元の位置に収め待機位置に集まります。
  • 点検・解散:
  • 身体や服装の点検を行い、指揮者の「点検報告」の号令で「3番員異常なし」と報告します。
  • 終了報告中は基本の姿勢で待ち、「わかれ」の号令で挙手注目の敬礼を行い解散します。

3番員における特に重要な注意事項（審査・減点ポイント）

  1. 伝達位置の厳守: 第2線延長時や放水中止時の4番員への伝達は、必ず「伝令停止線内」で行う必要があります。伝達線外での伝達は3点の大きな減点となります。
  2. とび口の構え方・姿勢: 破壊地点でのとび口の構えは、左手で柄を持つ位置が中央からずれても、左腕が水平である必要があります。また、右手の位置（後端からおおむね10cm残す）が不適切だと減点（とび口姿勢不適）となります。
  3. 吸管控綱の結着: ポンプ車への吸管控綱の結着は「もやい結び及び半結び」で行い、端末を約10cm残す必要があります。これらが不確実だと減点対象となります。
  4. ホース延長の確実性: 第2線延長時のホース展張や結合、おおむね2メートルの余裕ホースの確保など、基本的なホース延長要領が不確実な場合、細かく減点（各1点）が蓄積します。
