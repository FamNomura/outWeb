ポンプ車操法における2番員は、第1線延長時のホース（第1・第2ホース）の延長と放水開始の伝達、および注水補助を行い、続いて**第2線延長時の筒先・第3ホースの延長と火点への注水（放水）**を担う、非常に運動量が多く重要な役割を持っています。

実施手順と注意点について、時系列に沿って詳細に整理します。

1. 待機・乗車・下車

  • 待機: 待機位置（車両助手席側後部ドア横の乗車しやすい位置）に、火点方向を向いて整列休めの姿勢で待機します。
  • 点呼・乗車: 指揮者の「乗車」の号令で、そのまま右向け右をして速やかに乗車します（車両への乗り込みは三点支持を励行）。乗車後は車両の走行に対応できる姿勢をとります。
  • 下車: 4番員の「よし」の合図を確認した後、ドアを開放して下車します。
  • 【注意点】: 下車時のドア開放は窓から目視で後方確認後、二段操作で行い、勢い任せに閉めず最後まで片手を離さないようにします。

2. 第1線延長（ホース延長と伝達・注水補助）

  • ホースの搬送と展張・結合:
  • 下車後、ポンプ車後部に移動し、第2ホースを搬送に便利な位置（第1ホースの左右どちら側でも可）に置きます。
  • 第1ホースをめす金具が手前になるように展張し、おおむね2メートルの余裕ホースをとった後、めす金具を両手で持って第1放口に結合します。
  • 折りひざ（または準じた姿勢）で第2ホースを左肩に担ぎ、第1ホースのおす金具を右手に持ち腰につけて、展張ホースの左側に沿って延長します。この際、左手は体側から腰に上げる必要があります。
  • 第2結合場所で第1ホースを置き、第2ホースを展張して第1ホースと結合します。
  • 再び第2ホースのおす金具を右手に持ち腰につけて延長し、第3結合場所で第3ホース（1番員が展張済み）と結合します。
  • 火点方向に向きを変え、右足を左足に引きつけて基本の姿勢（両足かかとが第3結合部より火点側）をとります。
  • 放水開始の伝達と注水補助:
  • 1番員の「放水始め」の合図を復唱し、回れ右をして発進します。
  • 延長ホースの左側に沿って伝令停止線内の位置で4番員に相対して停止し、右手を垂直に挙げて「放水始め」と伝達します。
  • 4番員が復唱の手を下ろした後、回れ右をして最短距離で1番員の反対側1歩後方の注水部署に至り、「伝達終り」と呼唱して注水補助を行います。

3. 第2線延長（メインの役割への移行）

  • 第2線筒先・ホースの搬送と展張:
  • 1番員が指揮者の「第2線延長始め」を復唱したのち、「よし」と呼唱してホースを離し、伝令停止線内の位置まで戻って4番員に「第2線延長」と伝達します。
  • ポンプ車右側後部で4番員から筒先を受け取って背負い、ポンプ車後部のホース積載部から第3ホースを左肩に担いで、火点に向かって前進します。
  • 第1線と同様の要領で第3ホースを展張し、筒先を結合して基本注水姿勢をとります。
  • 放水開始と注水:
  • 3番員が第2ホースを第3ホースに結合したのを確認し、「第2線放水始め」と呼唱します。
  • 火点に向かっておおむね15メートル前進し、おおむね5メートルの余裕ホースをとり（後方に1メートルの注水補助場所を確保）、ノズルを開いて標的に注水します。

4. 放水中止

  • 放水停止: 指揮者の「放水止め」の号令で、ノズルを徐々に閉じて放水を中止し、「第2線放水止め」と呼唱します。
  • 排水操作:
  • 3番員が注水部署に戻り「伝達終り」と合図した後、筒先を右脇下に抱え、ノズルを下に向けて全開にし、排水します。
  • 合図によりノズルを完全に閉め、左手をプレイパイプ上部に、右手でノズルを握って「よし」と呼唱し、筒先を右足ぎわに立てて姿勢を正します。

5. 収納・点検・解散

  • 収納:
  • 指揮者の「おさめ」の号令に「よし」と呼唱し、筒先をホースから離脱して背負います。
  • 指揮者とともに発進し、第2線第1ホースをまたいでポンプ車左側を通り、筒先を元の位置に収納して待機位置に集まります。
  • 点検・解散:
  • 身体や服装の点検を行い、指揮者の「点検報告」の号令で「2番員異常なし」と報告します。
  • 終了報告中は基本の姿勢で待ち、「わかれ」の号令で挙手注目の敬礼を行い解散します。

2番員における特に重要な注意事項（審査・減点ポイント）

  1. 伝達位置の厳守: 4番員への伝達は必ず「伝令停止線内」で行う必要があります。伝達線外での伝達は3点の大きな減点となります。
  2. 注水補助の姿勢と位置: 注水補助部署（1番員の反対側1歩後方）を確実にとり、反動力に耐える自然な前傾姿勢（膝を地面に着けない）をとる必要があります。適切に行われないと減点対象です。
  3. 筒先位置（放水停止線）の厳守: 第2線延長時、放水停止線を越えて筒先を構えたり足を踏み越えたりすると、3点の大きな減点となります。
  4. ホース延長・展張の確実性: 各ホースの搬送要領、展張要領（右足で押さえ、前方へ転がすなど）、結合確認などが不確実だと細かく減点（各1点）が蓄積してしまいます。
  5. 行動開始の時機: 第2線延長時、1番員の「第2線延長始め」の復唱前に行動を開始してしまうと3点減点となるため、必ず復唱を聞いてから「よし」と呼唱して離れる必要があります。
