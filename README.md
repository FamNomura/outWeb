
## プロジェクト名称
OutWeb

## プロジェクト概要
本プロジェクトは、特定のディレクトリ（リポジトリ内）にHTMLファイルや静的アセット（CSS、JavaScriptなど）を格納するだけで、Webブラウザを介して自動的に外部公開・配信する静的Webサイトホスティングシステムです。
複雑なサーバー構築やデプロイ設定を一切行うことなく、ファイルベースの管理のみで即座にドキュメントやWebページを外部の閲覧者に届ける仕組みを目的としています。

## 主要な技術スタック
- **フロントエンド（配信コンテンツ）**: HTML5, CSS3, JavaScript (ES6+)
- **データフォーマット**: JSON (クライアントサイドでのキーワード検索インデックス用)
- **フォントサービス**: Google Fonts (Noto Sans JP)

## ファイル一覧と解説

### ルート・管理

* **README.md**
* 本プロジェクトの取扱説明書、およびリポジトリ全体の概要を記述するためのマークダウンファイルです。



### プラットフォーム・共通静的ファイル

* **souhotaikai2026/index.html**
* プラットフォームのメインハブ（ポータル画面）です。公開されているすべてのコンテンツへのリンク一覧を内包し、ユーザーの入り口となる責務を持ちます。


* **souhotaikai2026/search_index.json**
* サイト内の全ドキュメントのタイトル、URL、本文テキストを構造化した検索用インデックスデータベースです。


* **souhotaikai2026/static/search.js**
* ポータル画面の検索ボックスと連動し、`search_index.json` を非同期で読み込んでリアルタイムなキーワードフィルタリング（インクリメンタルサーチ）を行うフロントエンドスクリプトです。


* **souhotaikai2026/static/style.css**
* システム全体（ポータルおよび各詳細ドキュメント）に適用される共通デザインシートです。スマートフォンの表示に最適化されたレスポンシブデザインと、はみ出し防止処理（`overflow-wrap`、横スクロール対応等）の責務を担います。



### 自動公開ドキュメントコンテンツ（格納されたHTML群）

*※本プロジェクトの自動公開システムによって配信されている実際のドキュメントファイル群です。*

* **00_指揮者/index.html** : 指揮者の役割、時系列手順、審査・減点ポイントの解説。
* **01_１番員/index.html** : 1番員の筒先搬送、ホース展張、注水姿勢、減点ポイントの解説。
* **02_２番員/index.html** : 2番員のホース延長、放水伝達、第2線筒先搬送の解説。
* **03_３番員/index.html** : 3番員の吸管伸長・投入、とび口搬送・構えの解説。
* **04_４番員/index.html** : 4番員のポンプ車機関操作、送水圧力管理、コック開閉の解説。
* **10_ポンプ車操法の全体的な審査/index.html** : 行動・計時・総合審査の配点と採点基準の解説。
* **11_全体の動き/index.html** : タイムスケジュールに沿った全隊員の連携動作と重要ポイントの解説。
* **99_ファイル一覧/index.html** & **doc/yoryo.pdf** : 大会公式の実施要領・審査要領PDFを配信するファイルマネージャー。

## エントリーポイント

本プロジェクトのWebアプリケーションとしての開始地点（エントリーポイント）は以下の通りです。

1. **初期アクセス**: 閲覧者がホスティングURL（Webサイトのルート）にアクセスすると、最初のエントリーポイントとして `souhotaikai2026/index.html` が読み込まれます。
2. **スクリプト・スタイルのロード**: 画面表示と同時に `static/style.css` でスマートフォン・PC双方のレイアウトが最適化され、`static/search.js` が起動します。
3. **検索機能の初期化**: `search.js` 内の非同期処理によって `search_index.json` が即座にフェッチされ、メモリ上に検索用データが展開されます。
4. **ナビゲーションとドキュメント遷移**: ユーザーが検索または一覧から各フォルダ（例: `00_指揮者/index.html`）へのリンクをクリックすることで、完全に独立した子ドキュメントへとシームレスに遷移・動作します。
