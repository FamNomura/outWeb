1. 調味料・油脂類（余計なものが入っていない基本調味料）

  • ウエストゴールド グラスフェッドバター（有塩・無塩）
    ニュージーランド産の放牧飼育牛のミルクから作られたバターです。一般的なスーパーでは高価なグラスフェッドバターが、業スーでは直輸入価格で手に入ります。保存料・着色料無添加で、栄養価が高いのが特徴です。
  • ブルガリア産 純粋はちみつ
    45度以上で加熱されていないため、はちみつ本来の栄養素が残っていると評判です。250gで500円を切る圧倒的コスパで、砂糖代わりの天然甘味料として重宝します。
  • 菊川の塩こうじ
    原材料が「米こうじ、食塩、酒精」のみとシンプル。肉や魚を柔らかくするだけでなく、化学調味料を使わずに旨味を引き出せる万能調味料です。
  • オーガニック マスコバド糖
    精製されていないオーガニックの黒砂糖です。ミネラルが豊富で、お菓子作りや料理にコクを出したい時に最適です。

2. 冷凍食品（素材そのものの良さを活かしたストック）

  • オーガニック ブロッコリー（スペイン産）
    有機栽培されたブロッコリーが冷凍されており、カット済みなのでそのまま使えます。農薬が気になる方でも安心して買えるオーガニック商品です。
  • ミックスベリー / 冷凍みかん
    砂糖不使用で、果物そのものを急速冷凍しています。市販のアイスクリームを食べるよりもヘルシーなデザートになり、スムージーの材料としても人気です。
  • 冷凍アボカド（ダイスカット）
    原材料はアボカドと酸化防止剤（ビタミンCなど）のみ。手軽に良質な脂質を摂取でき、サラダや和え物に便利です。

3. 缶詰・瓶詰（原材料をチェックすべき隠れた名品）

  • オイルサーディン（モロッコ産）
    原材料が「いわし、大豆油、食塩」のみのものが多く、非常にシンプルです。タンパク質やカルシウム、EPA・DHAが手軽に摂れます。
  • ライトツナフレーク缶（水煮）
    油漬けではなく「水煮（食塩のみ）」タイプを選ぶと、余計な脂質を抑えられ、原材料も非常にクリーンです。
  • トリノで作ったトマト＆イタリアンチーズパスタソース
    イタリア直輸入で、水を一切使わず野菜の水分だけで作られています。保存料不使用で、本場のシンプルな美味しさが楽しめます。

4. ヘルシーなおやつ・穀物

  • 砂糖不使用 デーツバー
    デーツをベースにナッツやオーツ麦を固めたもので、砂糖不使用ながら満足感が高いです。忙しい時の補食や、お子様のおやつとしても安心です。
  • ローストココナッツチップス
    原材料が「ココナッツ、砂糖、食塩」のみ（※商品により異なりますが基本シンプルです）。ノンフライで香ばしく、食物繊維も豊富です。
  • トルコ産 ブルグルピラフ（ひよこ豆）
    「ブルグル」は全粒小麦を挽き割りにしたもので、白米よりも低GIで食物繊維が豊富です。温めるだけで食べられるレトルトタイプも、原材料がシンプルで体に優しいです。

1. 鮮度と産地にこだわった「素材重視」のメイン食材

  • 吉備高原若どり もも肉 / むね肉（冷蔵）
    岡山県産のブランド鶏で、一度も凍結させずに店舗に届くため鮮度が抜群です。ドリップが少なく、肉質が非常に柔らかいのが特徴。国産でこの品質・価格は、他のスーパーの追随を許さない業スーの目玉商品です。
  • 骨とりさば切身（冷凍）
    2025年から2026年にかけてさらに人気が定着したシリーズ。骨を丁寧に取り除いてあり、原材料は「さば」のみ（酸化防止剤不使用のものが多い）。調理が楽なだけでなく、お子様や高齢の方でも安心して食べられる素材重視の逸品です。
  • 大粒むきえび（冷凍）
    背わた処理済みで、加熱しても縮みにくいと評判です。保水剤（添加物）の使用が最小限に抑えられており、エビ本来の旨味とプリプリ感をこの価格で楽しめるのは驚異的です。

2. 砂糖・添加物を控えた「体に優しい」おやつ・軽食

  • 種抜きデーツ / デーツバー
    「天然のキャンディ」と呼ばれるデーツ（なつめやし）は、砂糖不使用・無添加。ミネラルや食物繊維が豊富で、ダイエット中や健康意識の高い層の定番です。デーツバーは持ち運びにも便利。
  • オーガニックチョコレート（カカオ70%）
    ベルギーなどから直輸入されるオーガニックチョコ。乳化剤不使用や、有機原料にこだわったものが100円〜200円台で見つかります。ハイカカオでポリフェノール摂取にも最適です。
  • こだわり極プリン
    原材料が「乳製品、卵、砂糖」と非常にシンプル。香料や着色料を使わず、素材の力だけで固めたプリンで、専門店のような素朴で濃厚な味わいが100円以下で楽しめます。

3. 料理の質を上げる「クリーンな」調味料・ペースト

  • ごまペースト（タヒニ）
    中東から直輸入される、ごま100%のペースト。乳化剤や油を足していないため、ごま本来の栄養と香りが凝縮されています。ドレッシングや和え物、パンに塗るなど汎用性が極めて高いです。
  • 粒マスタード（フランス産）
    原材料は「醸造酢、からし種、食塩」のみ。着色料や増粘剤が入っていないことが多く、プチプチした食感と本場の酸味が楽しめます。
  • アンチョビペースト（チューブ）
    缶詰と違い、使いたい分だけ出せる便利アイテム。原材料がカタクチイワシと塩、ひまわり油のみとシンプルで、化学調味料に頼らず料理に深いコクを出せます。

4. 時短と安全を両立する「冷凍素材」

  • 冷凍カットほうれん草 / インゲン（欧州産）
    ベルギーやスペインなど、農業大国の厳しい基準をクリアした野菜が急速冷凍されています。軽く湯通ししてあるだけなので、余計な味付けや保存料がなく、素材そのものの栄養をストックできます。
  • むき枝豆
    原材料は「えび（えだまめ）と塩」のみ。サラダや炊き込みご飯、お弁当の隙間に。タンパク質が手軽に補給できる優秀なストック食材です。

1. 穀物・主食（低GI・食物繊維重視）

  • オートミール（リトアニア産・ラトビア産など）
    1kg単位で数百円という、他店の半額近いコスパです。原材料は「オーツ麦」のみ。朝食や米化（ご飯の代わり）に最適で、添加物一切なしの全粒穀物として不動の人気です。
  • 全粒粉スパゲッティ（イタリア産）
    イタリア直輸入の全粒粉100％パスタ。精製されていないため食物繊維やビタミンが豊富です。独特の香ばしさがあり、ソースとの絡みも良く、健康志向の方に選ばれています。
  • オーガニック キヌア
    スーパーフードとして知られるキヌアも、業スーなら驚きの価格です。サラダに混ぜたり、ご飯と一緒に炊くだけで栄養価を底上げできます。

2. 植物性タンパク質・乳製品（原材料シンプル）

  • 100% ピーナッツバター（インド産など）
    原材料は「ピーナッツ」のみ。砂糖、塩、植物油脂すら入っていない純粋なペーストです。料理のコク出しや、はちみつと合わせてパンに塗るなど、自分好みの味に調整できるのが最大のメリットです。
  • オーツミルク（イタリア・スウェーデン産）
    砂糖不使用で、オーツ麦の自然な甘みを生かした植物性ミルク。一般的なスーパーでは1Lで500円近くしますが、業スーなら300円前後で見つかることもあり、乳製品を控えたい方の強い味方です。
  • ギリシャヨーグルト（または水切りヨーグルト）
    タンパク質が豊富で、余計な増粘剤や香料が入っていないタイプが人気です。濃厚なので満足感が高く、おやつ代わりにもなります。

3. 保存が利く「素材そのまま」缶詰・パウチ

  • ひよこ豆 / レンズ豆（缶詰・紙パック）
    原材料は「豆、水、塩」のみ。サラダやカレーにサッと入れるだけで、良質なタンパク質と食物繊維をプラスできます。イタリア産などの紙パック入りは、ゴミ出しも楽で人気です。
  • トマトピューレ（イタリア産・テトラパック）
    トマトを裏ごししただけのもので、食塩も無添加。トマト缶よりも酸味がまろやかで、料理のベースとして非常に優秀です。
  • ココナッツミルク（タイ産）
    漂白剤不使用（BPAフリー缶など）のものが選ばれています。カレーだけでなく、砂糖不使用のデザート作りにも重宝します。

4. 乾物・海藻（ミネラル補給に）

  • カットわかめ（韓国産・中国産など） / 焼き海苔
    乾燥野菜や海藻類は、水で戻すだけで添加物なしの栄養源になります。特にお徳用の焼き海苔は、おにぎりやサラダのトッピングに毎日気兼ねなく使えます。
  • 素焼きミックスナッツ
    「食塩不使用・油不使用」のタイプ。ローストしただけのナッツは、オメガ3脂肪酸やビタミンEの宝庫です。小腹が空いた時の「体にいいおやつ」として最強です。

5. 冷凍の「魚介・海鮮」

  • 殻付ムール貝（チリ産）
    原材料は「ムール貝」のみ。真空パックで冷凍されており、加熱するだけで豪華な一皿に。ダシもしっかり出るので、パスタやスープに使うと化学調味料なしで深い味わいになります。
  • 冷凍あさり（むき身）
    砂抜き済みで、原材料はあさりのみ。タウリンや鉄分が豊富で、味噌汁やパスタにパラパラと入れるだけで栄養バランスが整います。
